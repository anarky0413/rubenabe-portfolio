Cards

Cards group an image, title, description, and action into a reusable layout. This card style is used for project previews and selected work sections.

Project card

Reusable project preview

Use this card to present a project, case study, or selected work item. The image gives visual context, while the title, description, and button guide the user toward the next page.

Accessibility considerations

Card accessibility

The card uses readable contrast, clear spacing, and a direct action button. The image includes descriptive alt text, and the button label makes the next action clear for users moving to another page or project view.
