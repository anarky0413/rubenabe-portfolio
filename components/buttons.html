Buttons

The button system includes primary, secondary, disabled, and text link styles. Each style supports a different level of action priority while keeping the same visual language.

Primary button

Primary

Use primary buttons for the most important action on a page or section. This style uses a dark brown fill with cream text for strong contrast.

Go somewhere

Button HTML

<a class="demo-button demo-button-primary" href="#">Go somewhere</a>

Button CSS

.demo-button-primary {
  background: var(--brown);
  color: var(--cream);
  border: 1px solid var(--brown);
  box-shadow: 0 8px 18px rgba(0, 0, 0, 0.18);
}

.demo-button-primary:hover,
.demo-button-primary:focus {
  background: var(--cream);
  color: var(--brown);
  border-color: var(--brown);
  box-shadow:
    inset 0 4px 12px rgba(80, 61, 13, 0.26),
    0 4px 10px rgba(0, 0, 0, 0.08);
}

Secondary button

Secondary

Use secondary buttons for supporting actions that should still feel available, but not as visually dominant as the primary action.

Go somewhere

Button HTML

<a class="demo-button demo-button-secondary" href="#">Go somewhere</a>

Button CSS

.demo-button-secondary {
  background: var(--cream);
  color: var(--brown);
  border: 1px solid var(--brown);
  box-shadow: inset 0 3px 10px rgba(80, 61, 13, 0.16);
}

.demo-button-secondary:hover,
.demo-button-secondary:focus {
  background: var(--brown);
  color: var(--cream);
  border-color: var(--brown);
  box-shadow: 0 8px 18px rgba(0, 0, 0, 0.16);
}

Disabled button

Disabled

Use disabled buttons when an action is visible but unavailable. The muted grey styling shows that the control cannot currently be selected.

Button HTML

<button class="demo-button demo-button-disabled" disabled>Go somewhere</button>

Button CSS

.demo-button-disabled {
  background: #e2e2e2;
  color: #b5b5b5;
  border: 1px solid #bdbdbd;
  cursor: not-allowed;
  box-shadow: inset 0 3px 10px rgba(0, 0, 0, 0.14);
}

Text link

Text Link

Use text links for lower emphasis actions, inline movement, or page navigation that does not need a filled button treatment.

Go somewhere

Button HTML

<a class="demo-text-link" href="#">Go somewhere</a>

Button CSS

.demo-text-link {
  color: var(--text);
  font-size: 16px;
  font-weight: 800;
  text-decoration: underline;
  text-underline-offset: 5px;
  transition: color 180ms ease;
}

.demo-text-link:hover,
.demo-text-link:focus {
  color: var(--brown);
}
