Navigation

The portfolio navigation uses the same brand language as the design system, but the links are organized for a personal portfolio: Home, About, Works, and Resume.

Portfolio navigation

Responsive navigation bar

This navigation pattern is intended for the final portfolio pages. It keeps the logo on the left and places the main portfolio sections on the right. On smaller screens, the links collapse into a simple menu icon.

Accessibility considerations

Navigation accessibility

The navigation uses readable link labels, strong color contrast, a clear active state, and an accessible menu button for smaller screens. The logo includes alt text, and the navigation is labeled so screen reader users can understand its purpose.
