Usage

Color application

These notes explain how the palette should be applied across pages and components.

Primary usage

  • Use Nav Brown for primary buttons, dark navigation areas, and important interface controls.
  • Use Navigation Cream for logo marks, hover states, and soft contrast over dark brand areas.
  • Use Text Dark for headings, paragraphs, labels, and most readable content.
  • Use Background Light for the main page surface to keep the system clean and open.

Gradient usage

  • Use Gradient Nav for navigation bars, footers, and high identity areas.
  • Use Gradient Card for portfolio cards, project previews, and visual content blocks.
  • Do not overuse gradients in long text areas because they can reduce readability.
  • Pair gradients with solid white or light surfaces to keep the page balanced.
